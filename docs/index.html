Overview

Bezie provides an interface for creating complex automation shapes with ease. Parameters can be easily mapped from Bezie to any DAW and can be changed in realtime.

Features

Automation

Automate up to eight parameters simultaneously in a single grid

Snap to grid

Snap to both X and Y coordinates

Resolution

Up to 128th note grid resolution

Inversion

Vertical and horizontal inversion

Automatic Updates

Updates will be downloaded automatically and installed on quit
