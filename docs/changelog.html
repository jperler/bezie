v1.3.7

  • Introduced modes (clock & controller)

In clock mode (existing/default functionality), Bezie transmits automation against your DAW's MIDI clock. Controller mode allows you to trigger Bezie automation with either a hardware MIDI controller or your computer's keyboard (shortcuts below). In controller mode you can easily map from your hardware controller as well as select the tempo in which to send out the automation. These modes can be selected in MIDI preferences.

  • Keyboard shortcuts added:
    • shift + 1-8: Trigger envelope (active in controller mode only)
    • up/down: Cycle through curve types on the selected point (default, bezier 1, bezier 2)
    • backspace: Delete selected point

  • Settings icon added for MIDI preferences
  • Full MIDI CC channel selection (channels 1-119) per envelope
  • MIDI CC device presets (Save/Load Preset in MIDI preferences)
  • Envelope display names (Name envelopes in MIDI preferences)
  • Selected envelope's display name or channels name will now display below grid
  • Keyboard shortcuts added:
    • 1-8: Envelope switching
    • command (mac) / ctrl (win) + c: Copy envelope
    • command (mac) / ctrl (win) + v: Paste envelope
    • command (mac) / ctrl (win) + 1: Decrease note interval
    • command (mac) / ctrl (win) + 2: Increase note interval
    • command (mac) / ctrl (win) + -: Zoom out
    • command (mac) / ctrl (win) + =: Zoom in
    • p: Power
    • v: Vertical inversion
    • h: Horizontal inversion
    • b: Broadcast MIDI channel for mapping (equivalent to clicking the horn)
    • t: Toggle triplet mode
    • s: Toggle snap to grid mode
